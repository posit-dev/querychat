Edit a file by replacing an exact string

Use this to make incremental changes to the data-description file as you learn
more, so the analyst can watch it evolve. Replaces a single exact occurrence of
`old` with `new`.

The `old` string must match the file exactly and appear exactly once; if it is
missing or ambiguous, the edit fails and nothing is changed. Read the file first
if you are unsure of its current contents, and include enough surrounding
context to make `old` unique.
